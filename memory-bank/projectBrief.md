# Project Brief: Original_AI Ecosystem

## Mission
Build a self-evolving AI agent system supported by a modular, hybrid memory infrastructure.

## Core Components
1. **Original_AI**: The primary autonomous agent.
2. **LogicHive**: The library of reusable "Intelligence Units".
3. **SharedMemoryServer**: The persistent "Long-term Memory" hub.

## Goals
- **Zero-Knowledge Core**: Keep the agent lightweight by offloading knowledge and skills to MCP servers.
- **Persistence**: Ensure context and learned skills survive across sessions.
- **Synchronization**: Sync context between different tools (Cursor, VS Code, etc.).
