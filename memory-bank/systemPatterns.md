# System Patterns

## Hybrid Memory (Graph + Bank)
Combined structured data (SQLite) and unstructured context (Markdown) to cater to both machine-readable queries and human/LLM-readable summaries.

## Self-Anatomy / Self-Evolution
The process where an agent identifies its own modular, high-quality logic and saves it to a persistent vault (LogicHive) for reuse in future versions or other projects.

## Path Hardening
Environment-injected absolute paths for all storage-backed MCP servers to ensure stability in multi-environment configurations (Windows, Linux, Docker).
